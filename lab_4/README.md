# Android додаток для розрахунку струму короткого замикання

## Опис

Мобільний додаток для Android, який виконує розрахунки параметрів короткого замикання в електричних мережах.

## Функціональність

Додаток дозволяє:
- Вводити параметри електричної мережі (Ucn, Sk, Uk, Snom_t)
- Виконувати розрахунки опорів та струму короткого замикання
- Відображати результати в зручному форматі

## Формули розрахунків

- **Xc** = (Ucn²) / Sk
- **Xt** = (Uk × Ucn²) / (100 × Snom_t)
- **X_all** = Xc + Xt
- **Ipo** = Ucn / (√3 × X_all)

## Дизайн

- Градієнтним фоном

## Вимоги

- Android Studio Hedgehog | 2023.1.1 або новіша
- Android SDK 24 (Android 7.0) або вище
- Kotlin 1.9.20+

## Встановлення та запуск

1. Відкрийте проект в Android Studio
2. Синхронізуйте Gradle файли
3. Запустіть на емуляторі або реальному пристрої
